sheep-and-sheepdog
==================

Graphical simulation of sheep and sheepdogs using Boids model

Authors:
  Jason Fu
  Yu Tomita

Requires:
  Python 2.5+
  pyOpenGL

To run:
  $ python sheep_dog_graphical.py
